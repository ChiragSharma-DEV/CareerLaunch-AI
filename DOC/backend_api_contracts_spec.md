## SECTION 1: BROWSER LOCALSTORAGE STATE CONTRACTS

To maintain state and pass candidate intelligence between pages, CareerLaunch AI stores data structures inside the browser's persistent `localStorage`.

### 1.1 `resume_text`
*   **Type:** `string`
*   **Role:** Raw text extracted from the PDF resume via PDF.js.
*   **Format:** Sanitized text string.

### 1.2 `candidate_profile`
*   **Type:** `JSON object`
*   **Role:** Candidate contact and parsed skills profile.
*   **Schema:**
```json
{
  "name": "string",
  "email": "string",
  "education": ["string"],
  "skills": ["string"],
  "projects": [],
  "experience": [],
  "certifications": []
}
```

### 1.3 `github_analysis`
*   **Type:** `JSON object`
*   **Role:** Ingested GitHub repository details fetched via the public GitHub API.
*   **Schema:**
```json
{
  "owner": "string",
  "repo": "string",
  "name": "string",
  "description": "string",
  "language": "string",
  "stars": 8,
  "repo_score": 75,
  "tech_stack": ["string"],
  "file_tree": ["string"],
  "repositories": ["string"],
  "verified_skills": ["string"],
  "commit_history": [
    {
      "sha": "string",
      "message": "string",
      "date": "string"
    }
  ]
}
```

### 1.4 `job_analysis`
*   **Type:** `JSON object`
*   **Role:** Target Job description and evaluated gaps generated by Groq.
*   **Schema:**
```json
{
  "job_description": "string",
  "match_score": 85,
  "matched_skills": ["string"],
  "skill_gaps": ["string"],
  "match_justification": "string",
  "learning_roadmap": {
    "[SkillName]": {
      "concept_to_learn": "string",
      "recommended_resource": "string",
      "estimated_hours": 6
    }
  }
}
```

### 1.5 `linkedin_analysis`
*   **Type:** `JSON object`
*   **Role:** Ingested and audited LinkedIn profile feedback scores and checklist items.
*   **Schema:**
```json
{
  "headline_score": 85,
  "headline_feedback": "string",
  "about_score": 75,
  "about_feedback": "string",
  "experience_score": 80,
  "experience_feedback": "string",
  "completeness_score": 90,
  "overall_score": 82,
  "profile_tips": ["string"],
  "extracted_skills": ["string"]
}
```

## SECTION 2: GROQ LLM API CONTRACTS

The following JSON payloads outline the request and expected response formats when invoking the Groq Chat Completions endpoint (`https://api.groq.com/openai/v1/chat/completions`) in JSON Mode.

### 2.1 Compatibility Gap Analysis API (Market Insights Page)

#### 1. POST Request Payload:
*   **Endpoint:** `https://api.groq.com/openai/v1/chat/completions`
*   **Headers:**
    *   `Content-Type: application/json`
    *   `Authorization: Bearer <user_groq_api_key>`
*   **Body:**
```json
{
  "model": "llama-3.1-8b-instant",
  "messages": [
    { "role": "system", "content": "You are an AI Technical Evaluator. Output valid JSON only." },
    { "role": "user", "content": "JOB DESCRIPTION: ... CANDIDATE RESUME: ... VALIDATED GITHUB METRICS: ..." }
  ],
  "temperature": 0.1,
  "response_format": { "type": "json_object" }
}
```

#### 2. Expected Output JSON Schema:
```json
{
  "match_score": 80,
  "matched_skills": ["Python", "FastAPI"],
  "skill_gaps": ["Docker", "Redis"],
  "match_justification": "Detailed matching justification text.",
  "learning_roadmap": {
    "Docker": {
      "concept_to_learn": "Multi-stage builds and compose configurations.",
      "recommended_resource": "Docker Deep Dive Guide",
      "estimated_hours": 8
    }
  }
}
```

### 2.2 Interview Coach Agent API (Mock Interview Page)

The `InterviewCoachAgent` V3.0 communicates with Groq LLaMA-3.1 to dynamically generate questions and evaluate candidate responses.

#### 1. Dynamic Question Generation Payload
*   **POST Body:**
```json
{
  "model": "llama-3.1-8b-instant",
  "messages": [
    { "role": "system", "content": "You are an AI Interview Coach. Output valid JSON only." },
    { "role": "user", "content": "<Context, preferences, category, difficulty, and session memory>" }
  ],
  "temperature": 0.3,
  "response_format": { "type": "json_object" }
}
```

*   **Expected Output JSON Schema:**
```json
{
  "question_id": 1,
  "category": "Skill",
  "topic": "React Rendering Optimization",
  "difficulty": "L2",
  "question": "Typewriter question text.",
  "follow_up_candidates": ["Follow-up option 1", "Follow-up option 2"],
  "answer_evaluation": {
    "expected_focus_areas": ["Virtualization", "useMemo"]
  },
  "memory_updates": {
    "covered_skills": ["React"],
    "current_topic": "React Rendering",
    "difficulty_level": "L2"
  }
}
```

#### 2. Answer Evaluation Payload
*   **POST Body:**
```json
{
  "model": "llama-3.1-8b-instant",
  "messages": [
    { "role": "system", "content": "You are an AI Interview Coach. Output valid JSON only." },
    { "role": "user", "content": "<Question details, candidate answer text, expected focus areas>" }
  ],
  "temperature": 0.2,
  "response_format": { "type": "json_object" }
}
```

*   **Expected Output JSON Schema:**
```json
{
  "score": 8,
  "answer_classification": "Strong",
  "strong_points": ["Point 1", "Point 2"],
  "gaps_in_answer": ["Gap 1"],
  "is_bluff_detected": false,
  "optimized_star_answer": "SITUATION: ... TASK: ... ACTION: ... RESULT: ...",
  "memory_updates": {
    "verified_skills": ["React"],
    "unverified_skills": [],
    "difficulty_level": "L3"
  }
}
```
